## 🎮 Usage Workflows

### Quick Study Session
1. Open `index.html`
2. Go to ⏱️ Timer
3. Click Start (25 min)
4. Study/take notes
5. Click quiz when ready
6. Review results

### Deep Learning Session
1. Create custom quiz ([QUIZ-CREATION-GUIDE.md](QUIZ-CREATION-GUIDE.md))
2. Load quiz in ❓ Quiz
3. Take notes (📝 Notes)
4. Review results
5. Check spaced rep queue (📊 Progress)
6. Retake quiz

### Long-term Study Plan
1. Create multiple quizzes
2. Study daily (build streak 🔥)
3. Track progress (📊 Progress)
4. Review difficult items (spaced rep)
5. Export data regularly (💾)
