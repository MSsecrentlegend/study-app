# Quick Start Guide - Lectures & Files Features

## 🚀 Getting Started

### First Time Setup

1. **Set Your Admin Password**
   - Click ⚙️ Settings button in top right
   - Scroll to "Set Admin Password:" section
   - Enter a secure password (remember it!)
   - Click "Set Password"

2. **Start Using Lectures Tab**
   - Click 🎥 المحاضرات (Lectures) tab
   - Click 🔐 "Admin Mode" button
   - Enter your admin password
   - Upload buttons will appear

3. **Start Using Files Tab**
   - Click 📁 Files tab
   - Click 🔐 "Admin Mode" button
   - Enter your admin password
   - Upload buttons will appear

---

## 📚 Uploading Lectures (Videos)

### Allowed Formats:
- ✅ `.mp4` (MPEG-4 Video)
- ✅ `.mov` (QuickTime Video)

### Steps:

1. Go to **Lectures** tab
2. Click **Admin Mode** button
3. Enter admin password
4. Select **Subject** from dropdown:
   - Arabic (اللغة العربية)
   - English (اللغة الإنجليزية)
   - Math (الرياضيات)
   - Science (العلوم)
   - History (التاريخ)
   - Philosophy (الفلسفة)
   - French (اللغة الفرنسية)
   - Programming (البرمجة)
   - Religion (التربية الدينية)

5. Click **Browse** and select .mp4 or .mov file
6. Click **Upload Video**
7. Success! Video appears in folder

### Video Organization:
```
📍 Arabic (اللغة العربية)
   - video1.mp4 (2.5 GB)
   - lesson-1.mov (1.8 GB)

📍 Math (الرياضيات)
   - calculus-intro.mp4
   - algebra-advanced.mp4
```

## 📄 Uploading Files (Documents & Images)

### Allowed Formats:
- ✅ `.pdf` (PDF Documents)
- ✅ `.jpg` / `.jpeg` (JPEG Images)
- ✅ `.png` (PNG Images)
- ✅ `.gif` (GIF Images)

### Steps:

1. Go to **Files** tab
2. Click **Admin Mode** button
3. Enter admin password
4. Select **Folder** from dropdown:
   - Same subjects as Lectures
   - **Plus Extra folder** for miscellaneous resources

5. Click **Browse** and select your file
6. Click **Upload File**
7. Success! File appears in folder

### File Organization:
```
📁 Science (العلوم)
   - physics-notes.pdf (2.5 MB)
   - biology-diagram.png (1.2 MB)

📁 Extra
   - resources.pdf
   - helpful-image.jpg
```

## 🔐 Security & Admin Mode

### Why Admin Mode?
- Prevents accidental file deletion
- Restricts uploads to authorized users
- Keeps data organized and manageable

### Changing Admin Password:
1. Go to Settings ⚙️
2. Enter new password in "Set Admin Password"
3. Click "Set Password"
4. Old password is replaced

⚠️ **Important**: If you forget your password, you'll need to clear your browser data to reset it.

## 🆘 Troubleshooting

### "Admin Password Not Set" Error
- Go to Settings ⚙️
- Set an admin password
- Try again

### "Upload Failed" Error
- Check file format (.mp4/.mov for videos, .pdf/.jpg/.png for files)
- Check file size isn't too large
- Try refreshing the page

### Password Not Working
- Make sure Caps Lock is OFF
- Check you entered correct password
- If stuck: Clear browser data → Set new password

### Files Not Appearing After Upload
- Refresh the page
- Make sure you clicked the correct folder dropdown
- Check browser console for errors

### Mobile Upload Issues
- Use Portrait orientation for best experience
- Make sure you have good internet connection
- Try smaller files first

## 💡 Tips & Best Practices

### Naming Files:
- Use clear, descriptive names
- Include course or topic name
- Use hyphens instead of spaces
- Example: `arabic-03-grammar-rules.mp4`

### Organizing Subjects:
- Use main subject folders for primary content
- Use "Extra" folder for supplementary resources
- Keep related files together

### Video Best Practices:
- Use H.264 codec for better compatibility
- 720p or 1080p resolution recommended
- Keep file sizes under 2GB for better performance
- Use AAC audio for best quality

### File Management:
- Use consistent naming convention
- Organize by date or topic
- Regular backups of important files
- Delete old/unused files regularly
